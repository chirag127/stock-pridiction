stock-pridiction

Simple AI‑powered stock price prediction built by Chirag Singhal, Backend & GenAI Specialist.

Learn More About the Author

What It Does

  • Fetches latest market data from Yahoo Finance.
  • Trains a Linear Regression model on historical prices.
  • Predicts the next trading day's closing price.
  • Easy to run locally with a single command.

Try It Live

Run the script locally or explore the source code. No deployment needed – just clone and execute.

git clone https://github.com/chirag127/stock-pridiction.git
cd stock-pridiction
pip install -r requirements.txt
python stock_prediction.py --ticker AAPL
